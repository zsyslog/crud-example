# crud-example

CRUD example using Express, Nodejs, Angularjs, Bootstrap y Elasticsearch

# API 

<strong>GET - view user data</strong>
```
/crud/api/user/:id
```

<strong>POST - new user</strong>
```
/crud/api/user/:id
```

<strong>PUT - existing user</strong>
```
/crud/api/user/:id
```

<strong>DELETE - remove user</strong>
```
/crud/api/user/:id
```
